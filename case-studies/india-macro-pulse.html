India Macro Pulse — Data Release Intelligence

A Streamlit-powered intelligence platform for institutional economists and fixed-income traders tracking Indian macro releases.

Deployed MacroeconomicsPython Claude APIStreamlit Data Engineering

Context

For economists and traders, macro data releases are high-stakes moments. When MOSPI or the RBI releases CPI or IIP numbers, the first 60 seconds of interpretation can determine the day's alpha. However, raw releases often sit in PDFs or unformatted HTML, requiring manual extraction and calculation to understand the "surprise" relative to consensus.

The **India Macro Pulse** was built to bridge this gap, transforming a static data release into an interactive, AI-synthesized intelligence brief within seconds of publication.

System Architecture

The system is built as a highly responsive ingestion and analysis pipeline:

  • Automated Ingestion — Scrapers built with BeautifulSoup4 and pdfplumber monitor MOSPI and RBI index pages. As soon as a release hits, the system fetches the document and normalizes the raw numbers into a queryable SQLite store.
  • Economic Decomposition — Beyond the headline number, the engine breaks down CPI into its critical components: food, fuel, and core. It also handles the fiscal and calendar adjustments necessary for a clean comparison.
  • Surprise Indexing — The system automatically benchmarks the release against consensus forecasts, calculating a "Surprise Score" that flags whether the movement is statistically significant for market positioning.
  • AI Synthesis — Using the Anthropic Claude API, the system generates an expert-level flash brief. It doesn't just say what the number is; it analyzes the drivers and stance, providing a senior-economist-style briefing for rapid assimilation.

The Tech Stack

Built with efficiency in mind: Python for the core logic, Streamlit for the UI, and SQLite for lightweight, reliable persistence.

The frontend is designed for split-second readability. Economists can toggle between historical trends, component-level heatmaps, and the AI synthesis window. The entire stack is containerized for deployment, ensuring that the scraping agents run with high uptime and low latency.

Status

The first version of the system is fully operational and deployed. It currently tracks CPI and IIP releases, with specialized modules for RBI MPC communications and External Sector data in the pipeline.
